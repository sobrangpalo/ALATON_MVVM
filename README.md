# ALATON_MVVM

## Overview

ALATON_MVVM is a project that demonstrates the MVVM design pattern in mobile applications.

## Getting Started

To get started with the project, follow these instructions:

1. Clone the repository: `git clone https://github.com/sobrangpalo/ALATON_MVVM`
2. Open the project in your preferred IDE.
3. Run the application on your device or emulator.
