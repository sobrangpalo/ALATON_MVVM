# ALATON_MVVM

## Overview

ALATON_MVVM is a mobile application project that demonstrates the MVVM (Model-View-ViewModel) design pattern. It was built using AIAppStudio entirely on an iPad, making it an impressive effort in mobile-first development.

## Product Flow

This application follows a simple product flow:

1. **User Authentication**:
   - Sign up or log in to the app.
2. **Product Browsing**:
   - View and explore items available for purchase.
3. **Add to Cart**:
   - Add products to your cart.
4. **Order Processing**:
   - Checkout and track orders.

## Technologies Used

- **Programming Language**: Kotlin
- **Framework or Platform**: MVVM Design Pattern
- **Development Software**: AIAppStudio (mobile-based development platform)

## Prerequisites

To work with this project, you need the following:

1. A PC, Mac, or iPad
2. Android Studio (for PC/Mac deployment)

## Step-by-Step Instructions to Run in Android Studio

To run the project in Android Studio, follow these steps:

1. **Clone the Repository**:
   ```
   git clone https://github.com/sobrangpalo/ALATON_MVVM
   ```
2. **Open Android Studio**:
   - Start Android Studio.
   - Select "Open an existing project" and navigate to the cloned repository folder.
3. **Install Dependencies**:
   - Allow Android Studio to install any missing SDK dependencies.
4. **Connect an Emulator or Device**:
   - Set up and connect a virtual or physical device.
5. **Run the Application**:
   - Click the "Run" button in Android Studio to start the app on your selected device.

## Note for iPad Developers

Since this project was developed entirely on an iPad using AIAppStudio, adjustments have been made to enable later deployment through Android Studio. AIAppStudio allows building and exporting code for inspection and refinement by a broader development team.
